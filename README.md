# Anchor AMM

Automated Market Maker implementation on Solana using Anchor framework.

## Overview
This project implements an Automated Market Maker (AMM) that allows users to trade tokens through liquidity pools with algorithmic price discovery.

## Features
- Liquidity pool creation and management
- Token swapping functionality
- Liquidity provision and withdrawal
- Dynamic pricing based on pool ratios

## Technologies
- Rust
- Anchor
- Solana CLI
- SPL Tokens

## Getting Started
1. Install Solana CLI and Anchor
2. Run `anchor build` to build the program
3. Run `anchor test` to run integration tests
4. Deploy using `anchor deploy`
